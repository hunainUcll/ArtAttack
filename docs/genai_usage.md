# GenAI Usage Overview

GenAI tools were used during this project to support planning, explanation, and debugging.

## How GenAI was used

- To structure the project plan and divide the work into realistic steps.
- To explain concepts such as Stable Diffusion, ControlNet, Canny preprocessing, and LoRA fine-tuning.
- To help generate initial Python code for the ControlNet pipeline and Gradio interface.
- To help write and improve the documentation.
- To prepare possible oral defense questions and answers.

## What the team did manually

- Tested the code in the development environment.
- Selected sketches and prompts for evaluation.
- Interpreted the generated results.
- Adjusted parameters such as seed, guidance scale, inference steps, and ControlNet scale.
- Prepared the final repository and demonstration.

## Reflection

GenAI was useful as a support tool, especially for understanding unfamiliar libraries and creating a first working structure. However, the final implementation still required testing, debugging, interpretation, and project-specific decisions by the team.
